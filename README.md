# Segmentasi Produk Fashion Berdasarkan Pola Penjualan Bulanan
## Menggunakan Metode K-Means dan Hierarchical Clustering

Aplikasi ini dibuat sebagai bagian dari tugas akhir (skripsi) dengan tujuan untuk melakukan
segmentasi produk fashion berdasarkan pola penjualan setiap bulan. Proses segmentasi dilakukan
menggunakan dua metode clustering, yaitu **K-Means** dan **Hierarchical Clustering**.

## Latar Belakang
Dalam dunia bisnis fashion, memahami pola penjualan setiap produk sangat penting untuk
menentukan strategi pemasaran, pengelolaan stok, dan perencanaan produksi. Dengan metode
clustering, produk dapat dikelompokkan berdasarkan kemiripan pola penjualan bulanannya
sehingga memudahkan pengambilan keputusan.

## Tujuan Penelitian
Mengelompokkan produk fashion ke dalam beberapa segmen berdasarkan pola penjualan bulanan
menggunakan metode:
- K-Means Clustering
- Hierarchical Clustering

Hasil segmentasi ini diharapkan dapat membantu perusahaan dalam:
- Menentukan strategi promosi
- Mengatur stok barang
- Mengidentifikasi produk paling laku dan kurang laku
